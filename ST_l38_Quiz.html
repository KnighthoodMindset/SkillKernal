Software testing - lecture 37 - Quiz

Why is mutation testing considered a stronger testing criterion compared to graph-based and logical predicate testing?





Mutation testing verifies that mutants are executed, cause internal state changes, and propagate to affect outputs, making it stronger than local criteria.

Which model explains the strict conditions that mutation testing imposes to kill mutants?





Mutation testing relies on reachability, infection, and propagation to ensure that mutations affect program output.

How does mutation testing subsume node coverage in graph-based testing?





Bomb statements are used to verify that each node in the control flow graph is executed at least once.

Which mutation operators help mutation testing subsume edge coverage?





Relational operator replacements flip decision predicates, ensuring both true and false branches are executed.

Which logical coverage criteria are subsumed by mutation testing?





Mutation testing inherently covers predicate, clause, and generalized active clause coverage by flipping conditions.

Why does mutation testing not subsume correlated active clause coverage (CACC)?





Mutation testing typically modifies one clause at a time and cannot enforce simultaneous control required by CACC.

How does mutation testing subsume the all-definitions (all-defs) criterion in data flow coverage?





Removing a variable definition ensures that its absence leads to errors, satisfying all-defs coverage.

Which testing criteria are NOT subsumed by mutation testing?





Mutation testing does not subsume complex criteria like prime path, complete path, combinatorial coverage, CACC, or RACC.

Which coverage model uses the insertion of bomb statements to confirm execution of specific elements?





Bomb statements are used in mutation testing to verify node coverage by confirming specific nodes are visited.

Why is mutation testing considered a global testing criterion?





Mutation testing is global as it ensures that executing mutants causes state changes and propagates differences to outputs.
