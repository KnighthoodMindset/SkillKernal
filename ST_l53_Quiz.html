Software testing - lecture 53 - Quiz

What is the main focus of object-oriented software testing?





Object-oriented software testing focuses on ensuring correct interactions between classes, methods, and objects, especially during integration testing.

Which features of object-oriented software increase testing complexity?





Abstraction, inheritance, and polymorphism complicate visualization of interactions and fault detection in object-oriented software.

What are coupling variables in object-oriented integration testing?





Coupling variables are those defined in one method or instance and used in another, representing potential interactions between methods.

How can coupling occur in object-oriented programs?





Coupling occurs directly within instances or indirectly when methods use other methods, with execution paths affected by polymorphism.

What is a polymorphic call set?





A polymorphic call set includes all methods that may execute through a given instance depending on its runtime type.

What does the All Coupling Sequences (ACS) criterion require?





ACS requires at least one test case to execute each coupling path between methods.

How does the All Polyclasses (APC) criterion extend ACS?





APC extends ACS by ensuring at least one test case is executed for every object type an instance may be bound to, addressing polymorphism.

What is the requirement of the All Coupling Definitions and Uses (ACDU) criterion?





ACDU requires that every last definition of a coupling variable reaches at least one corresponding first use.

Which coverage criterion is the most comprehensive for object-oriented integration testing?





The most comprehensive criterion considers every last definition to every first use across all coupling variables, including inheritance, polymorphism, and object bindings.

Why is systematic coverage of coupling and polymorphism important in object-oriented testing?





Systematic coverage ensures detection of faults caused by object-oriented features such as inheritance, polymorphism, and interactions among methods and objects.
