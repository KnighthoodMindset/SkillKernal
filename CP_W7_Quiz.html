Cybersecurity - Overall Quiz

__________ is the principle that an individual is entrusted to safeguard and control equipment, keying material, and information and is answerable to proper authority for the loss or misuse of that equipment or information.





Correct Answer: Accountability

Which of the following rings provides the least privileges in an Operating System?





Correct Answer: Ring 3

Which of the following is not true about asymmetric encryption?





Correct Answer: Asymmetric encryption relies on asymmetric cryptography, also known as private key cryptography.

Which team is responsible for defending against cyber threats by implementing security measures and monitoring systems for suspicious activities?





Correct Answer: Blue Team

True or False : Attribution, the process of identifying the perpetrators of cyber attacks, is easily achievable in cyberspace.



Correct Answer: False

What does the "Command and Control" stage of the Intrusion Kill Chain involve?





Correct Answer: Establishing a communication channel to remotely access the compromised system.

Which layer of the OSI model provides reliable data transfer and error control of connections?





Correct Answer: Transport layer

MSQ: Which of the following statements about ethical hacking is/are correct ? (Select all that apply. More than one answer may be possible)





Correct Answers: Ethical hackers operate with explicit permission from the organization; Ethical hackers communicate findings to the organization; Ethical hacking is a proactive measure for security posture.

Which of the following statement is true?





Correct Answer: TCP is connection-oriented, and UDP is connectionless

What is the primary function of a Demilitarized Zone (DMZ) in network security architecture?





Correct Answer: Create an isolated buffer zone for public-facing services, separating them from the internal network
