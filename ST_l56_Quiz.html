Software testing - lecture 56 - Quiz

What is the primary purpose of regression testing?





Regression testing ensures that software modifications or bug fixes do not break existing functionality.

What is the term used for selecting a subset of existing test cases to test modified software?





Selecting a subset of existing test cases to test modified software is called regression test selection.

Which set of test cases is created specifically to cover modifications not addressed by the selected subset?





T double prime represents newly created test cases covering modifications not addressed by the selected subset T prime.

What problem does creating a combined set of test cases for future regression testing address?





Maintaining a combined set of test cases (T triple prime) addresses the test suite maintenance problem for future regression testing.

Which regression test selection technique ensures that all faults in the modified program are detectable?





Safe regression technique ensures that selected test cases detect all faults present in the modified program.

Which approach involves executing every test case from the original suite to ensure coverage?





The retest-all approach executes every test case from the original set to ensure full coverage.

What factors are considered when selecting test cases for regression testing?





Regression test selection relies on system knowledge, focusing on defect-prone areas, frequently modified code, and critical features.

Which tools are commonly used for automated regression testing?





Automated regression testing tools include Regression Tester, TimeShiftX, and Micro Focus tools for legacy systems.

What is the main advantage of record-and-playback tools in regression testing?





Record-and-playback tools automate execution of previously run or newly created test cases, verifying expected outcomes consistently.

Why must obsolete test cases be removed when specifications change during regression testing?





Obsolete test cases must be removed because they are invalid for the modified program and could lead to incorrect results.
