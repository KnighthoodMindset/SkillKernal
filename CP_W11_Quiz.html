Cybersecurity - Overall Quiz

Within how many hours does GDPR require organizations to report cyber incidents?





Correct Answer: 72 hours

True or False: Privacy is a fundamental right in India.



Correct Answer: True

Which legal basis for processing personal data applies when someone's life is in danger, and the processing is necessary to save it?





Correct Answer: Vital interests

GDPR applies to which of the following organizations?





Correct Answer: Any organization, regardless of location, that processes the personal data of individuals in EU

Which of the following is/are guiding principles of the GDPR? (Select all that apply. More than one answer may be possible)





Correct Answers: Lawfulness, fairness, and transparency; Purpose limitation; Data minimization

Which of the following is not a valid legal basis for processing personal data under GDPR?





Correct Answer: Profit maximization

True or False: GDPR doesn't mandate the exact security measures organizations must take.



Correct Answer: True

Which of the following best describes the concept of data localization in India?





Correct Answer: Mandatory storage and processing of certain data within the country’s borders

Under GDPR, which of the following is a key responsibility of the Data Protection Officer (DPO)?





Correct Answer: Advising on and monitoring firm’s compliance with data protection laws

In the context of Aadhaar data, who would be considered the Data Principal?





Correct Answer: The individual to whom the Aadhaar number and related data belong
