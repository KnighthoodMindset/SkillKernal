Software testing - lecture 21 - Quiz

Why are logical predicates important in software testing?





Logical predicates control decision statements such as if, while, for, and do-while, influencing execution paths.

What does propositional logic deal with?





Propositional logic deals with Boolean atomic propositions that evaluate to true or false.

Which Boolean connectives are sufficient to define all propositional formulas?





The NOT and OR operators are sufficient to express all propositional formulas.

How can the implication “alpha implies beta” be expressed using NOT and OR?





“Alpha implies beta” is equivalent to “NOT alpha OR beta” in propositional logic.

What is the satisfiability problem in propositional logic?





Satisfiability checks if there exists an assignment of truth values that makes a propositional formula true.

How does predicate logic extend propositional logic?





Predicate logic adds variables, functions, and relations, allowing expression of program conditions.

What is a clause in predicate logic?





A clause is a Boolean atomic predicate, an individual component in a predicate that contains no Boolean operator.

Why is satisfiability undecidable in general for predicate logic?





Predicate logic can include variables with infinite domains, making general satisfiability undecidable.

Which tools can assist in checking satisfiability for practical predicate logic cases?





SAT solvers and SMT solvers help check satisfiability in practical predicate logic testing scenarios.

What is the primary focus in testing predicates during program execution?





Testing focuses on verifying whether a given assignment of values makes the predicate true during execution.
