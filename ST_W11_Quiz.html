Software testing - Overall Quiz

State yes or no: Is concolic testing a white-box testing technique?



Yes. Concolic testing is a white-box testing technique that combines concrete execution with symbolic execution to systematically explore program paths.

Which of the following is true about concolic testing?



Concolic testing simultaneously maintains concrete state for execution and symbolic state for path constraint analysis.

What is the use of a SAT/SMT solver in symbolic testing?





SAT/SMT solvers are used to solve symbolic path constraints and generate concrete input values that will follow the desired execution paths.

State true or false: Symbolic execution can be used to detect non-termination in programs.



False. Symbolic execution cannot reliably detect non-termination because it does not execute all paths indefinitely; it primarily focuses on exploring feasible paths symbolically.

Which of the following is a list of techniques used in the algorithm deployed by DART?





DART combines directed search, random testing, and constraint solvers to automatically explore program paths and generate test inputs.

Which of the following strategy is used for input search in concolic testing?



Concolic testing uses a combination of systematic and random search interleaved with path-sensitive symbolic analysis to explore program execution paths efficiently.
