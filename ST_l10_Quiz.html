Software testing - lecture 10 - Quiz

1. What does it mean when one structural coverage criterion subsumes another?





One criterion subsumes another when every test that satisfies the first also satisfies the second.

2. Which of the following correctly represents the order of subsumption among common coverage criteria?





Prime path coverage subsumes edge coverage, which in turn subsumes node coverage.

3. Why does edge coverage subsume node coverage?





Edge coverage subsumes node coverage because each edge connects two nodes, ensuring node visits.

4. What is a side trip in graph-based testing?





A side trip is a temporary deviation from the main path that rejoins it at the same vertex.

5. How does a detour differ from a side trip?





A detour may skip certain intermediate nodes or edges before rejoining the main path.

6. What best defines a prime path?





A prime path is a maximal simple path that cannot be extended without repeating vertices.

7. Why is graph visualization important in coverage analysis?





Graph visualization helps identify loops and branches to derive meaningful test paths.

8. Which algorithms are useful for constructing test paths for node and edge coverage?





BFS and DFS algorithms are used to systematically explore nodes and edges for coverage.

9. What distinguishes direct coverage from indirect coverage?





Direct coverage means the simple path appears as a contiguous subpath; indirect allows side trips.

10. What is the main advantage of using prime path coverage in testing?





Prime path coverage achieves a balance between comprehensive testing and practical feasibility.
