Software testing - Overall Quiz

Which of the following represents a basic block in a control flow graph?



Correct Answer: A basic block of statements is a sequence of statements such that if the first statement in the sequence is executed then all the statements in the sequence will also be executed.

Consider a variable count of type int. Suppose there is a statement count++ in a method. Which of the following statements are correct regarding the data flow definition of count?





Correct Answer: The statement is both a definition and use of count.

Consider a variable x of type double. Suppose there is a statement if(Math.log(x) >= 4.2) in a method. Is this considered a definition or use of x?



Correct Answer: The statement is a use of x.

State true or false: Consider a variable x in a program. Not every definition of x will always reach a use.



Correct Answer: True.

Which of the following best defines a du-path for a variable x?





Correct Answer: A du-path is a simple path from a definition of x to a use of x without any further definitions of x in-between.

State yes or no: We group du-paths with respect to a variable by their definitions.



Correct Answer: Yes.

Is it true that the all-du-paths data flow coverage criterion subsumes prime path coverage?



Correct Answer: No.

Which of the following statements are true when comparing traditional source code coverage criteria with graph-based coverage criteria?



Correct Answer: Node and statement coverage are the same, edge and branch coverage are the same.

Which of the following defines a linearly independent path of execution in a control flow graph?





Correct Answer: A path which does not contain other paths within it.

State true or false: Node and edge coverage, as test requirements for structural coverage over graphs, are given with the input graph itself.



Correct Answer: True.
