Software testing - Overall Quiz

Which of the following features of static websites are tested using graphs?



Graphs are used to test static websites by verifying that all links are correctly connected and identifying any dead links in the structure.

While testing web applications, which of the following is true regarding client-side vs. server-side testing?



Client-side and server-side are distinct, and testers often cannot access server code, limiting testing mainly to the client side.

State yes or no: It is possible to generate all the input screens that are passed to a web application.



It is not possible to generate all input screens because web applications can have dynamic forms and inputs that depend on previous user actions or server responses.

Which of the following is a list of techniques that can be used to provide test values to a web application?





Test values can be provided using random selection, data from user sessions, or values constructed according to the application's domain to simulate realistic usage scenarios.

Which of the following features can be checked using HTML without the need to write extra scripting constraints?





HTML provides features such as specifying transfer mode, enforcing input length limits, and validating input against a predefined list of values without additional scripting.

Is a control graph model useful for testing system level properties of web applications?



Control graph models are not suitable for system-level testing of web applications as they are more focused on component or module-level interactions rather than end-to-end behavior.

Which of the following is a complete list of operators that are used in component expressions?





Component expressions can include sequencing, iteration, selection, and aggregation to fully represent component behavior.

Which of the following defines the nodes of a yo-yo graph?



In a yo-yo graph, each method—new, inherited, or overridden—is represented as a node to analyze method-level interactions.

Which of the following best defines the edges of a yo-yo graph?





Edges in a yo-yo graph represent actual method calls and potential calls blocked due to overriding, capturing all call possibilities.

State true or false: All the faults in object-oriented programs can be determined statically.



False. Not all faults can be detected statically in object-oriented programs; some require dynamic analysis due to runtime behavior and polymorphism.
