Software testing - Week 7 - lecture 7 - Quiz

What is the primary use of the Breadth First Search (BFS) algorithm in software testing?





BFS is used to traverse graphs systematically and generate minimal test paths that satisfy coverage criteria.

Which of the following graph representations is more efficient for sparse graphs?





Adjacency lists are efficient for sparse graphs since they store only existing edges.

In a directed graph represented using an adjacency list, how many times does each edge appear?





In a directed graph, each edge appears only once in the adjacency list.

What data structure does BFS primarily use to explore vertices?





BFS uses a queue to store vertices to be explored in breadth-first order.

What color is assigned to a vertex when it has been discovered but not fully explored in BFS?





A gray vertex indicates it has been discovered but not fully explored.

What does the Breadth First Tree (BFT) produced by BFS represent?





The Breadth First Tree represents the shortest paths from the source vertex to all reachable vertices.

What is the time complexity of the BFS algorithm?





BFS runs in O(V + E) time since each vertex and edge is processed once.

Which of the following properties ensures that BFS finds the shortest path in unweighted graphs?





BFS explores vertices level by level, ensuring shortest path discovery.

What does the Queue Order Lemma state in BFS?





The Queue Order Lemma ensures that vertices are explored in non-decreasing order of their distance from the source.

What happens when BFS encounters an unconnected graph with multiple components?





When BFS encounters unconnected components, it restarts from another source, producing a forest of trees.
